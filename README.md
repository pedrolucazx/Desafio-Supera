## Como rodar o projeto

Clone esse projeto e acesse a pasta.

```bash
$ git clone https://github.com/pedrolucazx/Desafio-Supera.git
$ cd Desafio-Supera
```
Para iniciá-lo, siga os passos abaixo:
```bash
# Instalar as dependências
$ yarn
# Iniciar o projeto
$ yarn dev
```
